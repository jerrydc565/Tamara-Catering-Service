## ✨ Features

- 🍴 **Menu Showcase** — Display food categories, dishes, and catering packages
- 📋 **Services Section** — Highlight different catering services offered (events, delivery, etc.)
- 📸 **Gallery** — Visual showcase of past catering events and dishes
- 📝 **Booking / Enquiry Form** — Customers can reach out and request catering services
- 📍 **Location & Contact** — Address, phone, and contact details
- 📱 **Fully Responsive** — Mobile-friendly design across all devices
- ✨ **Smooth Animations** — Clean transitions and hover effects

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/jerryDev2/Tamara-Catering-Service.git

# Navigate into the project
cd Tamara-Catering-Service

# Install dependencies
npm install

# Start development server
npm run dev
```

Open your browser at `http://localhost:5173`

### Build for Production

```bash
npm run build
```
